# Diseño, distribución y posición


## display

Define el comportamiento del elemento.

``` css
div {
  display: flex;
}
```


## justify-content

Alineación horizontal en flexbox.

``` css
.container {
  justify-content: center;
}
```


## align-items

Alineación vertical en flexbox.

``` css
.container {
  align-items: center;
}
```


## flex

Sistema de diseño flexible.

``` css
.container {
  display: flex;
}
```


## grid

Sistema de diseño en cuadrícula.

``` css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}
```


## position

Controla la posición del elemento.

``` css
.box {
  position: relative;  /* absolute, fixed */
}
```
