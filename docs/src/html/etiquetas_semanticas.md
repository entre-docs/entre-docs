# Etiquetas Semánticas

## header

Define la cabecera de una página o sección.

``` html
<header>
  <h1>Mi sitio web</h1>
</header>
```

## nav

Contiene enlaces de navegación.

``` html
<nav>
  <a href="#">Inicio</a>
  <a href="#">Contacto</a>
</nav>
```

## main

Representa el contenido principal.

``` html
<main>
  <p>Contenido principal del sitio.</p>
</main>
```

## section

Agrupa contenidos relacionados.

``` html
<section>
  <h2>Noticias</h2>
  <p>Últimas novedades.</p>
</section>
```

## article

Contenido independiente como artículos o publicaciones.

``` html
<article>
  <h2>Título del artículo</h2>
  <p>Contenido del artículo.</p>
</article>
```

## aside

Contenido complementario o lateral.

``` html
<aside>
  <p>Publicidad o enlaces relacionados.</p>
</aside>
```

## footer

Pie de página.

``` html
<footer>
  <p>© 2026 Mi sitio web</p>
</footer>
```
