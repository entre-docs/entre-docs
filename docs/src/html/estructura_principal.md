# Etiquetas HTML

Las etiquetas HTML son los bloques que forman la estructura de una página web.


## html

Etiqueta raíz que engloba todo el documento HTML.

``` html
<html>
    <!-- AQUI VA EL CÓDIGO -->
</html>
```

## head

Contiene metadatos, enlaces a estilos y scripts.

``` html
<head>
  <title>Mi página</title>
</head>
```

## body

Contiene el contenido visible de la página

``` html
<body>
  <h1>Hola mundo</h1>
</body>
```
