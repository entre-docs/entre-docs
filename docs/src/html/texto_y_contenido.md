# Texto y contenido

## h1 a h6

Encabezados de distintos tamaños.

``` html
<h1>Título principal</h1>
<h2>Subtítulo</h2>
```

## p

Define un párrafo.

``` html
<p>Este es un párrafo de ejemplo.</p>
```

## a

Crea un enlace.

``` html
<a href="https://ejemplo.com">Visitar sitio</a>
```

## strong

Texto importante en negrita.

``` html
<p>
    <strong>Importante:</strong> guardar cambios.
</p>
```

## em

Texto en cursiva.

``` html
<p>
    <em>Texto destacado</em>
</p>
```

## img

Insertar imágenes.

``` html
<img src="imagen.jpg" alt="Descripción de la imagen">
```
