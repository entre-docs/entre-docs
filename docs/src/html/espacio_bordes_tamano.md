# Espacio, bordes y tamaño


## margin

Espacio exterior.

``` css
div {
  margin: 20px;
}
```


## padding

Espacio interior.

``` css
div {
  padding: 15px;
}
```

## border

Borde del elemento.

``` css
div {
  border: 2px solid black;
}
```

## width / height

``` css
img {
  width: 200px;
  height: 150px;
}
```
