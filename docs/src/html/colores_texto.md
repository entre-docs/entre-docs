# Colores y Texto


## color

Define el color de texto.

``` css
p {
  color: blue;
}
```

## background-color

Color de fondo.

``` css
body {
  background-color: lightgray;
}
```

## font-size

Tamaño de la fuente.

``` css
h1 {
    font-size: 32px;
}
```

## font-family

Tipo de letra,

``` css
body {
  font-family: Arial, sans-serif;
}
```

## text-align

Alineación de texto.

``` css
p {
  text-align: center;
}
```
