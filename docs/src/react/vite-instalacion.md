# Crear Proyecto

## Vite: React + TypeScript

::: code-group

```bash [NPM]
$ npm create vite@latest
```

```bash [Yarn]
$ yarn create vite
```

## package.json
```json
{
  "scripts": {
    "dev": "vite", // dev server
    "build": "vite build", // build 
    "preview": "vite preview" // vista previa local: production build
  }
}
```
