# Identity as a Service (IDaaS)

Identity as a Service (IDaaS) permite a las empresas externalizar la gestión de usuarios y las funcionalidades de autenticación, autorización y gestión de accesos a un proveedor de servicios en la nube.

Los servicios de IDaaS ofrecen una variedad de funcionalidades, que pueden incluir:

1. Autenticación multifactor (MFA)
2. SSO (Single Sign-On) para acceder a múltiples aplicaciones con una sola credencial de inicio de sesión
3. Gestión centralizada de usuarios y grupos
4. Provisionamiento y desprovisionamiento de usuarios
5. Políticas de seguridad y acceso basadas en roles
6. Auditoría y generación de informes de actividades de usuarios
7. Integración con directorios corporativos existentes (por ejemplo, Active Directory)

<br>

::: tip ¡Importante!
Gracias a la implementación de IDaaS el desarrollo de componentes frontend se simplifica en gran medida, dado que ya no es necesario construir una pantalla de login, con todos los recaudos de seguridad que ello conllevaba, y tampoco es necesario desarrollar capas de backend para ir a obtener y validar usuarios, todo es realizado por la nube.
:::
